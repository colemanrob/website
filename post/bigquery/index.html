Using R with Google BigQuery

Posted on 2017-12-23

Google’s BigQuery has some interesting public datasets and luckily there’s an R package that makes interacting with the data extremely easy. The bigrquery library let’s you use SQL or dplyr statements to query the data which is super helpful.

Simple Query

I took a look at the data from the US Census Bureau’s International dataset which provides projections on population indicators. You need to create a projectid in the bigquery interface before you start.

library(tidyverse)
library(hrbrthemes)
library(bigrquery)

project <- "YOUR_PROJECT_ID"
sql <- "SELECT * FROM
[bigquery-public-data:census_bureau_international.birth_death_growth_rates]"
rawdata <- query_exec(sql, project = project)

That will load the dataframe into your environment for analysis or visualizations.

rawdata %>%
  select(Country = country_name, Year = year, `Growth Rate` = growth_rate) %>%
  filter(Country %in% c('Canada', 'United States')) %>%
  filter(Year > 2010) %>%
  ggplot(aes(x=Year, y=`Growth Rate`, group=Country)) +
  geom_line(aes(color=Country), size=0.8) +
  theme_ipsum_rc() +
  labs(y="Growth Rate Percentage",
       title="Population Growth Rate Over Time",
       caption="Source: US Census Bureau")
