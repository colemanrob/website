Mapping 2016 Census Data

Posted on 2017-12-15

The 2016 Census has a lot of useful data to be examined. Luckily, there’s a great R package cancensus that let’s you access and map the data easily.

You can use get_census to download the table and geographic features you need.

    sault_income <- get_census(dataset='CA16', regions=list(CMA="35590"),
                        vectors="v_CA16_2397", level='DA', quiet = TRUE,
                        geo_format = 'sf', labels = 'short')

The dataframe is ready to be mapped using ggplot or leaflet if you want to be interactive.

    bins <- c(0, 25000,50000, 75000, 100000, 125000,Inf)
    pal <- colorBin("RdYlBu", domain = sault_income$v_CA16_2397, bins = bins)

    saultmap <- leaflet(sault_income) %>%
      addProviderTiles(providers$CartoDB.Positron) %>%
      addPolygons(fillColor = ~pal(v_CA16_2397),
                  color = "white",
                  weight = 1,
                  opacity = 1,
                  fillOpacity = 0.55) %>%
      addLegend("bottomright", pal = pal, values = ~sault_income$v_CA16_2397,
                title = "Total Household Income",
                labFormat = labelFormat(prefix = "$"),
                opacity = 1
      ) %>%
      setView(lng = -84.346090, lat = 46.521858, zoom = 12)
