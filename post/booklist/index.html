Tracking Books

Using R to Generate a booklist

Posted on February 2, 2019

I’ve been trying to use goodreads to track my reading progress and wondered how difficult it would be to recreate a list of my books here. Turns out, very easy. The following script quickly transforms the goodreads export file into a booklist in markdown that can be seen here.

Step 1

First export your goodreads activity and load it into R. I use clean_names from the janitor package - one of my favourite convenience functions.

library(tidyverse)

raw_goodreads <- read_csv("~/Downloads/goodreads_library_export.csv") %>% 
  janitor::clean_names()

Step 2

Next I tidy the dataset. I wanted to show my rating as ⭐️ using the emoji : star : and strrep seemed to do the trick. I also wanted to provide a link to the book via the title and some ugly paste code seemed to do the trick.

star <- "⭐️"

raw_goodreads %>% 
  filter(my_rating != 0) %>% #remove books i haven't rated
  filter(date_added >= "2018-01-01") %>% #only books since 2018
  mutate(star_rating = strrep(star, as.integer(my_rating)), # turn rating into stars
         linked_title = paste("[", title, "]", "(https://www.goodreads.com/book/show/", book_id, ")", sep = "")) %>% #add links
  select("Title" = linked_title, "Author" = author, "Rating" = star_rating) %>% 
  knitr::kable()

Step 3

Not much else! I added the markdown code to /page/books and now I can run the script whenever I want to update. Head over to the book list to check out the results.
