Mapping 2016 Census Data

Posted on December 15, 2017

The 2016 Census has a lot of useful data to be examined. Luckily, there’s a great R package cancensus that let’s you access and map the data easily. You can use get_census to download the table and geographic features you need. sault_income <- get_census(dataset='CA16', regions=list(CMA="35590"), vectors="v_CA16_2397", level='DA', quiet = TRUE, geo_format = 'sf', labels = 'short') The dataframe is ready to be mapped using ggplot or leaflet if you want to be interactive.
